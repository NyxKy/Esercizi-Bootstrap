Qual è il font di default di bootstrap?

Se il browser trova "Helvetica Neue", usa quello. Se non è disponibile, passa a Helvetica. Se nemmeno quello è disponibile, usa Arial. Infine, se nessuno dei precedenti è disponibile, usa un qualsiasi font sans-serif di sistema
